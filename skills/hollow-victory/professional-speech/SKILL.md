# Professional Speech Skill

> *Turn ideas into structured, persuasive narratives.*

## When to Use

Activate this skill when:

- User needs to prepare for a speech, presentation, or team meeting
- User asks to structure a strategic communication or pitch
- User has unstructured ideas and needs a compelling narrative arc
- User mentions "talking points", "speech", "meeting agenda", "strategic communication"

## Frameworks

### Pyramid Principle (Minto)
- **Answer first** — lead with the conclusion
- **Group and summarize** — MECE breakdown of supporting arguments
- **Logically order** — deductive or inductive reasoning flow

### SCQA Framework
- **Situation** — What everyone agrees on
- **Complication** — What changed or went wrong
- **Question** — The key question this raises
- **Answer** — Your recommendation

### Communication Types

| Type | Structure | Duration |
|------|-----------|----------|
| Team Strategy Session | Full agenda + talking points + discussion guides | 90-120 min |
| Client Pitch | SCQA → Storyline → Key Slides | 30-60 min |
| Leadership Update | Bottom-line up front → 3 key points → Ask | 15-30 min |
| Quick Standup | Progress → Blockers → Next steps | 5-10 min |

## Workflow

### Step 1: Context Gathering
Understand: audience, objective, time constraint, key message

### Step 2: Framework Selection
Match communication type to optimal framework

### Step 3: Structure Design
Build narrative arc with clear transitions

### Step 4: Content Generation
Write talking points, discussion guides, and backup materials

### Step 5: Delivery Notes
Add timing marks, emphasis cues, and Q&A preparation
